# Neovim キーマッピング一覧

Leader キー: `,`

## ウィンドウ操作

| キー        | モード | 説明                  |
| ----------- | ------ | --------------------- |
| `<C-h>`     | Normal | 左のウィンドウに移動  |
| `<C-j>`     | Normal | 下のウィンドウに移動  |
| `<C-k>`     | Normal | 上のウィンドウに移動  |
| `<C-l>`     | Normal | 右のウィンドウに移動  |
| `<C-Up>`    | Normal | ウィンドウの高さを +2 |
| `<C-Down>`  | Normal | ウィンドウの高さを -2 |
| `<C-Left>`  | Normal | ウィンドウの幅を -2   |
| `<C-Right>` | Normal | ウィンドウの幅を +2   |

## バッファ操作

| キー    | モード | 説明                   |
| ------- | ------ | ---------------------- |
| `<S-l>` | Normal | 次のバッファに切り替え |
| `<S-h>` | Normal | 前のバッファに切り替え |

## ファイル・一般操作

| キー | モード | 説明                                 |
| ---- | ------ | ------------------------------------ |
| `,w` | Normal | ファイルを保存                       |
| `,q` | Normal | ウィンドウを閉じる                   |
| `,h` | Normal | 検索ハイライトを消去                 |
| `,,` | Normal | 現在のファイルを再読み込み（source） |

## ファイルエクスプローラ (nvim-tree)

| キー | モード | 説明                                  |
| ---- | ------ | ------------------------------------- |
| `,e` | Normal | ファイルツリーの表示/非表示を切り替え |

## ファジーファインダ (Telescope)

| キー  | モード | 説明                                                 |
| ----- | ------ | ---------------------------------------------------- |
| `,ff` | Normal | ファイル名で検索                                     |
| `,fg` | Normal | ファイル内容をグレップ検索                           |
| `,fb` | Normal | 開いているバッファを一覧表示                         |
| `,fh` | Normal | ヘルプタグを検索                                     |
| `,fe` | Normal | ファイルブラウザを開く                               |
| `,fd` | Normal | 現在のファイルのディレクトリでファイルブラウザを開く |

## Git

| キー  | モード | 説明           |
| ----- | ------ | -------------- |
| `,gg` | Normal | lazygit を開く |

## ターミナル (ToggleTerm)

| キー  | モード | 説明                           |
| ----- | ------ | ------------------------------ |
| `,tf` | Normal | フローティングターミナルを開く |
| `,th` | Normal | 水平分割でターミナルを開く     |
| `,tv` | Normal | 垂直分割でターミナルを開く     |

## テキスト編集

| キー    | モード       | 説明                               |
| ------- | ------------ | ---------------------------------- |
| `<`     | Visual       | インデントを減らす（選択維持）     |
| `>`     | Visual       | インデントを増やす（選択維持）     |
| `<A-j>` | Visual       | 選択行を下に移動                   |
| `<A-k>` | Visual       | 選択行を上に移動                   |
| `J`     | Visual Block | 選択行を下に移動                   |
| `K`     | Visual Block | 選択行を上に移動                   |
| `p`     | Visual       | 貼り付け（レジスタを上書きしない） |

## LSP（Neovim 0.11 ビルトイン + カスタム）

| キー    | モード | 説明                     | 備考               |
| ------- | ------ | ------------------------ | ------------------ |
| `K`     | Normal | ホバードキュメントを表示 | 0.10+ ビルトイン   |
| `grn`   | Normal | シンボルをリネーム       | 0.11+ ビルトイン   |
| `grr`   | Normal | 参照一覧を表示           | 0.11+ ビルトイン   |
| `gra`   | Normal | コードアクションを実行   | 0.11+ ビルトイン   |
| `gri`   | Normal | 実装へジャンプ           | 0.11+ ビルトイン   |
| `gO`    | Normal | ドキュメントシンボル     | 0.11+ ビルトイン   |
| `<C-s>` | Insert | シグネチャヘルプを表示   | 0.11+ ビルトイン   |
| `,f`    | Normal | ファイルをフォーマット   | カスタム (lsp.lua) |

## 補完 (blink.cmp / super-tab プリセット)

| キー      | モード | 説明                            |
| --------- | ------ | ------------------------------- |
| `<Tab>`   | Insert | 補完候補を選択 / 次の候補に移動 |
| `<S-Tab>` | Insert | 前の補完候補に移動              |

## モーション (leap.nvim)

| キー            | モード           | 説明                   |
| --------------- | ---------------- | ---------------------- |
| `s{char}{char}` | Normal/Visual/Op | 2 文字検索ジャンプ     |
| `S{char}{char}` | Normal           | 他ウィンドウへジャンプ |

## 囲み操作 (nvim-surround)

| キー               | モード | 説明                     |
| ------------------ | ------ | ------------------------ |
| `ys{motion}{char}` | Normal | 指定範囲を囲み文字で囲む |
| `ds{char}`         | Normal | 囲み文字を削除           |
| `cs{old}{new}`     | Normal | 囲み文字を変更           |

## ユーザーコマンド

| コマンド         | 説明                                             |
| ---------------- | ------------------------------------------------ |
| `:Rename {name}` | 現在のファイルをリネーム（旧ファイルは自動削除） |
